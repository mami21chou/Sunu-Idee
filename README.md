
#  Sunu-Idées - Boîte à idées collaborative

##  Description

Sunu-Idées est une application web collaborative permettant aux utilisateurs de partager, modifier et supprimer des idées anonymement. Les idées sont organisées par catégories et persistent grâce au LocalStorage.

##  Fonctionnalités

-  **Créer** une idée avec titre, catégorie et description
- **Lire** toutes les idées sur le mur collaboratif
-  **Modifier** une idée existante
-  **Supprimer** une idée
- **Rechercher** une idée par mot-clé
-  **Persistance** des données avec LocalStorage
-  **Design responsive** avec Tailwind CSS
